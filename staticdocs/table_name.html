Database table name for a given prefix and salt.

Usage

table_name(prefix, salt)

Arguments

prefix
character. Prefix.
salt
list. Salt for the table name.

Value

the table name. This will just be "prefix_" appended with the MD5 hash of the digest of the salt.

Description

Database table name for a given prefix and salt.
